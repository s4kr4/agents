# Skills 一覧

`~/.claude/skills/` 配下のスキル一覧。

| スキル             | 用途                                                                                            |
| ------------------ | ----------------------------------------------------------------------------------------------- |
| `/tdd`             | TDD（テスト駆動開発）のベストプラクティス。Red-Green-Refactorサイクル、テストファースト開発     |
| `/ts-implement`    | TypeScript実装のベストプラクティス・コーディング規約                                            |
| `/ts-formatter`    | TypeScript/JS向けフォーマット・リント設定                                                       |
| `/oxlint`          | oxlint + @stylistic/eslint-plugin によるTS/JSリント・フォーマット設定（シングル・モノレポ対応） |
| `/py-implement`    | Python実装のベストプラクティス・コーディング規約                                                |
| `/py-formatter`    | Python向けフォーマット・リント設定（Black、flake8等）                                           |
| `/react-implement` | React実装のパターンとベストプラクティス                                                         |
| `/ui-test`         | UIテスト作成のベストプラクティス                                                                |
| `/api-test`        | バックエンドAPIテスト作成のベストプラクティス                                                   |
| `/linux-diag`      | Linux系OSのシステム不具合診断                                                                   |
| `/skill-feedback`  | スキルのフィードバックを対話形式で収集・保存する                                                |
| `/skill-improve`   | フィードバックを分析してSKILL.mdの改善案を提案・適用する                                        |
| `/insight-capture` | 作業中の知見をスキルとして保存・既存スキルに統合する対話フロー          |
